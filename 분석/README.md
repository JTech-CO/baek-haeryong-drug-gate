## 🔑 사건 한 줄 요약

> 2023년 말레이시아 마약조직이 인천·김해공항을 통해 필로폰을 **인편(바디패킹) 약 115~127kg + 화물(나무도마) 56kg** 규모로 밀반입한 사건에서,
> 백해룡 경정(당시 영등포서 형사과장)이 **인천공항세관의 조직적 조력 정황**과 **검찰·경찰 수뇌부의 수사 외압·은폐 정황**을 포착했으나,
> 브리핑 중단·사건 이첩·좌천을 거쳐 수사가 멈췄고, 검찰 주도 합동수사단이 **'혐의없음'으로 종결**하자
> 백 경정이 **5,400쪽 수사기록의 총론·해설**을 공개한 것이 이 자료집입니다.

|---|---|
| 확인된 국내 유통 필로폰 | 300kg 이상 |
| 마약 규모 (자료집 추산) | 약 5,860억 원 |
| 전체 수사기록 분량 | 약 5,400쪽 |
| 인편(바디패킹) 밀수 | 1.27일 24kg / 총 115.9~127.4kg (압수 12.2kg) |

| 주요 사건번호 | 인천지검 2023형제7362호 · 중앙지검 2023형제12530호 · 2023조39호 · 경찰청 2025-00423호 |
